# הפעלת GitHub Pages - הוראות מפורטות

## ⚠️ חשוב: GitHub Pages לא מופעל!

הקבצים כבר ב-GitHub, אבל צריך להפעיל את GitHub Pages ידנית.

## שלבים להפעלה:

### 1. לך להגדרות המאגר:
https://github.com/meireinat/ashdodManagers/settings/pages

### 2. תחת "Source" (מקור):
- בחר **Branch: main**
- בחר **Folder: / (root)** ← זה חשוב!
- לחץ על **Save**

### 3. המתן 1-2 דקות

### 4. הקישור יהיה:
```
https://meireinat.github.io/ashdodManagers/
```

או:
```
https://meireinat.github.io/ashdodManagers/index.html
```

## קבצים שצריכים להיות ב-root:
- ✅ index.html
- ✅ בוט הנמל החכם.mp3
- ✅ asdod_port_logo_official.png

## בדיקה:
לאחר ההפעלה, בדוק:
1. לך ל: https://meireinat.github.io/ashdodManagers/
2. אמור לראות את המצגת

## אם עדיין לא עובד:
1. בדוק שהקבצים קיימים ב: https://github.com/meireinat/ashdodManagers
2. ודא ש-GitHub Pages מופעל (Settings > Pages)
3. בדוק את הקונסול בדפדפן (F12) לראות שגיאות
