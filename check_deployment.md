# בדיקת פרסום GitHub Pages

## בעיות אפשריות ופתרונות:

### 1. GitHub Pages לא מופעל
**פתרון:**
- לך ל: https://github.com/meireinat/ashdodManagers/settings/pages
- תחת "Source":
  - בחר **Branch: main**
  - בחר **Folder: / (root)**
- לחץ **Save**
- המתן 1-2 דקות

### 2. הקישור הנכון:
```
https://meireinat.github.io/ashdodManagers/deploy/presentation_v2.html
```

או:
```
https://meireinat.github.io/ashdodManagers/deploy/index.html
```

### 3. אם עדיין לא עובד:
- בדוק את הקונסול בדפדפן (F12) לראות שגיאות
- ודא שכל הקבצים נדחפו ל-GitHub
- בדוק שהקבצים קיימים ב: https://github.com/meireinat/ashdodManagers/tree/main/deploy

### 4. בדיקת קבצים ב-GitHub:
